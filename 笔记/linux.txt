linux目录结构
bin:命令执行文件
boot:系统引导文件
dev:计算机内部硬件 cpu 内存
etc 系统环境文件
home 用户目录文件
lib 依赖文件
lib64 依赖文件
media 外设文件 鼠标 键盘
mnt 挂载文件
opt 软件文件
proc 内存映射文件
root 家文件
run 运行结束产生文件
srv 运行时产生文件
sys 系统更新之后 计算机 硬件 产生文件
tmp 临时文件 定期删除
usr 用户设置的文件
var 额外产生日志文件
